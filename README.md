# BingoXRPL Website

Official website for $BINGO memecoin on XRP Ledger.

## Structure

```
bingo-site/
├── assets/
│   ├── css/          
│   ├── js/           
│   └── images/       
├── index.html       
├── bingo-roller.html 
├── CNAME             
└── README.md        
```

## Features

- Fully responsive design
- WCAG 2.1 AA accessible
- Firebase-powered live bingo game
- SEO optimized
- Mobile-first architecture

## Tech Stack

- HTML5, CSS3, JavaScript (ES6)
- Firebase Realtime Database
- GitHub Pages hosting
- No build tools required
